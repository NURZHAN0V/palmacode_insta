# Документация проекта

## Идея продукта

Создать универсальный шаблон сайта, который **визуально напоминает ленту популярной социальной сети Instagram**, но предназначен **персональным портфолио одного специалиста**. Продукт рассчитан на **массовую продажу мастерам разных профессий**, поэтому дизайн должен быть **строгим и аккуратным**, чтобы выгодно подчеркнуть фотографии работ, чертежи или примеры проектов.

### Управление контентом

Управление сайтом должно быть **интуитивно понятным**: владелец без сложных инструкций и без привлечения технических специалистов может **легко размещать свои материалы**.

### Опыт посетителя (без барьеров)

**Обязательное условие:** у посетителей сайта **нет форм входа и создания аккаунтов**. Любой человек может **сразу открыть страницу**, **просмотреть все материалы** и **связаться с владельцем** через простые кнопки контакта. Сайт работает как **современная цифровая визитка** — без лишних препятствий к просмотру контента.

### Акценты

- **Для клиента:** быстро оценить квалификацию мастера по работам и проектам.
- **Для специалиста:** простое готовое решение для представления в интернете без лишней технической нагрузки.
