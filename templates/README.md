## Шаблон приложения на Nuxt 4

Этот каталог содержит шаблон пользовательского интерфейса и структуры проекта, который используется как визуальный и архитектурный референс для разработки приложения.

### Основные принципы

- Используется **Nuxt 4** с клиентским рендерингом (SSR отключён).
- Для стилей применяется **Tailwind CSS** и только цвета из палитры `tailwind.config.js`.
- Управление состоянием реализуется через **Pinia**.
- Вёрстка и компоненты должны соответствовать этому шаблону, при расхождениях приоритет у визуального референса из `templates`.

### Структура шаблона

- Базовая раскладка страниц (layout, header, footer, основная область контента).
- Примеры типографики (заголовки, текст, ссылки).
- Шаблоны форм и элементов ввода (поля, кнопки, чекбоксы, переключатели и т.д.).
- Примеры модалок, дропдаунов и прочих интерактивных компонентов.

### Правила использования

- При создании новых страниц и компонентов ориентируйся на существующие примеры из `templates`.
- Все стандартные HTML‑контролы ввода (input, select, textarea и др.) заменяй на кастомные компоненты, визуально совпадающие с шаблоном.
- Сохраняй единый визуальный стиль: отступы, размеры, состояния (hover, focus, disabled и др.).
- Обеспечивай доступность (a11y): поддержка клавиатуры, корректный `tabindex`, `aria`‑атрибуты для сложных компонентов.

### Связанные документы
