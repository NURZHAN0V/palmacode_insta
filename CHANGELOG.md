# Журнал изменений

## [Unreleased]

### Исправлено

- Обновление Nuxt до 4.4.x: в dev больше не сыпется ложный «unhandled» на `GET /_nuxt/` (исправление обработки 404 в dev‑сервере, см. nuxt/nuxt#34225).

### Изменено

- Модалка поста: строка **Тема / Темы** (подписи из подборок по `themeIds`).
- Посты: метрика **просмотры** вместо лайков и комментариев; в модалке фото в пропорции **4:5** с `object-cover` (без полос).
- Шапка: логотип «PortfolioInsta» ведёт на главную (`NuxtLink`).
- Шапка: вместо ссылки на Telegram — кнопка «Поделиться» (`navigator.share` или копирование URL).
- Вкладки «Публикации» / «Услуги» / «Товары» / «Обо мне»: переключение контента; подборки-темы только на «Публикации»; `PortfolioServices` и `PortfolioGoods` (данные `services` / `products` в сторе); на «Обо мне» — `PortfolioAbout`, данные в `about`.
- Подборки-темы: пункт «Все публикации» (иконка сетки, сброс фильтра), затем темы; градиент у активного пункта; повторный клик по теме снимает фильтр; неактивные — стекло + hover.
- Модалка просмотра работы: каркас на белом фоне, слева тёмная зона под фото (`bg-zinc-950`), справа лёгкий градиент `white → teal-50 → warm-100`; кнопка закрытия и разделители подстроены под светлую панель.
- Лейаут: убраны фоновые декоративные «стеклянные» фигуры (воспринимались как пустые квадраты).
- Шапка: убран поиск по работам; слева бренд, справа иконки «домой», Telegram и аватар.
- В шаблонах используются стандартные имена автоимпорта Nuxt: `PortfolioNav`, `PortfolioProfile`, `PortfolioHighlights`, `PortfolioGrid` (файлы в `components/portfolio/` — `Nav.vue`, `Profile.vue`, …).

### Добавлено

- Клиентская главная страница портфолио (Nuxt, без бекенда): сетка работ, профиль, подборки, поиск по подписям, модалка просмотра.
- Визуал в стиле glassmorphism на градиентном фоне (палитра teal/warm, шрифты Comfortaa/Roboto).
- Pinia-стор `usePortfolioStore` с демо-данными и локальной фильтрацией поиска; демо-услуги и товары для вкладок «Услуги» и «Товары».
